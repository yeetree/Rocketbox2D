# Rocketbox2D and Engine

## Building
* Clone repo and submodules: `git clone --recurse-submodules https://github.com/yeetree/Rocketbox2D.git`
* Install Vulkan SDK from [https://vulkan.lunarg.com/sdk/home](https://vulkan.lunarg.com/sdk/home)
   * Make sure to install Vulkan Memory Allocator (VMA) and slangc from the installer as well
* Create a build directory `mkdir build && cd build`
* Configure `cmake ..` and build `cmake --build .`
* You can find deployed applocations in `path/to/build/bin`

Rocketbox2D and Engine are in development.
Engine is not production ready.
